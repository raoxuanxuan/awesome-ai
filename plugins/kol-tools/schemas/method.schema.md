# Method Page Schema

Use for `wiki/methods/*.md`.

## Purpose

A method page captures a stable analysis framework repeatedly used by the KOL.

## Required Sections

```text
# <Method>

## Core Rule
## Applies When
## Does Not Apply When
## Signals
## Failure Conditions
## Related Sources
## Related Positions
## Evidence
```

## Evidence Rules

- A new method requires repeated evidence or unusually strong explicit evidence.
- Each method rule must cite tweet ids.
- Distinguish explicit KOL statements from inferred framework summaries.

## Update Rules

- Strengthen existing methods before creating new methods.
- Add failure conditions whenever the KOL names a disconfirming signal.
- Avoid promoting one-off reactions into durable methodology.
